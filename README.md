# Restaurant Landing Page Template

A modern, responsive landing page template for restaurants, burger shops, and small food businesses.  
Easy to customize and ready for your own content.

## Features
- Fully responsive design (mobile-friendly)
- Sections for Home, Menu, FAQ, and Contact
- Simple HTML, CSS, and JavaScript (no backend)
- Easy to adapt for different restaurants or franchises
